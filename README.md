

Tic Tac Toe Game Database Collections 

   

1. **Users Collection:** 

   - Fields: 

     - userID (String) 

     - username (String) 

     - email (String) 

  

2. **GameRooms Collection:** 

   - Fields: 

     - roomID (String) 

     - player1ID (String) 

     - player2ID (String) 

     - currentPlayer (String) 

     - gameState (String) 

  

3. **Moves Collection:** 

   - Fields: 

     - moveID (String) 

     - roomID (String) 

     - playerID (String) 

     - position (int) 

     - timestamp (Timestamp)
